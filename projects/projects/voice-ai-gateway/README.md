# Voice AI Gateway

An AI-powered voice processing gateway that transforms audio input into structured actions.

## What it does
- Receives voice input (audio file or stream)
- Transcribes speech to text (ASR)
- Detects user intent
- Generates a contextual response
- Returns a voice or text output

## Why this project
This project demonstrates how modern AI can be applied to real-time voice operations such as IVRs, call centers, and automated assistants.
